# CLAUDE.md

Personal website for Albertina Durante — Astro 6 + React 19 + Panda CSS, deployed to Cloudflare Workers.

## Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Dev server (Panda CSS watch + Astro) |
| `npm run build` | Production build → `dist/` |
| `npm run format` | Auto-format with Prettier |
| `npm run lint` | ESLint + Prettier check |
| `npm run test` | Jest test suite |
| `npm run test:coverage` | Jest with 80% coverage enforcement |
| `npm run clean:build` | Remove `dist/`, `coverage/`, `.astro/` |
